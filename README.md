## ⚙️ Features

- 🔐 User Authentication (Login / Signup)
- 📦 TanStack Query for State Management
- 🌐 API Integration (Express + MongoDB backend)
- 📱 Responsive Layout for Android
- ☁️ Cloudinary image uploads
- 🔔 Push Notifications (optional)

---

## 🛠️ Tech Stack

- React Native (Expo)
- Redux
- React Navigation
- Axios
- Cloudinary
- Express.js
- MongoDB
